# 滤镜网站开发计划

## 📋 项目概述

一个纯前端的滤镜网站，内置 1000+ 潮流滤镜模板，用户上传照片后选择滤镜即可生成带滤镜效果的照片并导出。无需后端、无需登录，打开即用。

---

## 🏗 技术选型

| 层面 | 技术 | 理由 |
|------|------|------|
| 构建工具 | **Vite** | 极速开发体验，构建产物为纯静态文件 |
| 前端框架 | **原生 HTML/CSS/JS** | 零依赖，部署简单，打开即用 |
| 图片处理 | **Canvas API + CSS Filter** | 纯前端实现，无需后端，性能好 |
| 滤镜引擎 | **CSS filter 组合 + Canvas 像素处理** | CSS filter 负责基础调色（亮度/对比度/饱和度/色相/色温等），Canvas 负责叠加层、暗角等复杂效果 |
| 部署 | 纯静态文件，任意静态服务器即可 | GitHub Pages / Vercel / Netlify / OSS |

### 滤镜参数模型

每个滤镜由以下参数组合定义：

```js
{
  id: 'kr-001',                    // 唯一标识
  name: '首尔清晨',                  // 中文名称
  nameEn: 'Seoul Morning',         // 英文名称
  category: 'korean',              // 所属分类
  tags: ['清新', '日常', '高亮'],    // 标签（用于搜索）
  css: {                           // CSS filter 参数
    brightness: 1.15,              // 亮度 (0-3)
    contrast: 1.05,                // 对比度 (0-3)
    saturation: 1.20,              // 饱和度 (0-3)
    sepia: 0.08,                   // 褐色/怀旧 (0-1)
    hueRotate: 5,                  // 色相旋转 (deg, -360~360)
    blur: 0,                       // 模糊 (px)
  },
  overlay: {                       // Canvas 叠加层（可选）
    type: 'color',                 // 'color' | 'gradient' | 'vignette' | 'grain'
    color: 'rgba(255,235,210,0.04)', // 叠加颜色
    opacity: 0.3,                  // 叠加层不透明度
  },
  vignette: {                      // 暗角（可选）
    intensity: 0.15,               // 强度 (0-1)
    feather: 0.5,                  // 羽化程度 (0-1)
  },
  intensity: 1.0,                  // 默认滤镜强度
}
```

### 滤镜生成策略

1. **手工精调核心滤镜**：设计 ~200 个高质量核心滤镜，覆盖各大分类的代表性风格
2. **程序化生成变体**：基于核心滤镜，通过微调参数（亮度 ±0.05、饱和度 ±0.1、色相 ±3° 等）生成变体，扩充至 1000+
3. **保证质量**：每个生成的滤镜都经过视觉验证规则，确保效果合理不过度

### 关键 UI 特性

1. **拖拽上传区**：支持拖拽图片到上传区或点击选择文件
2. **滤镜缩略图**：每个滤镜显示在用户上传图片上的小预览
3. **分类标签横向滚动**：平滑滚动，高亮当前选中分类
4. **滤镜强度滑块**：可调节滤镜效果强度（0%~200%）

7. **响应式设计**：桌面端左右布局，移动端上下布局

## ⚙️ 核心功能模块

### 1. 图片上传模块 (`ui/uploader.js`)
- 支持格式：JPG、PNG、WebP、HEIC（转换为 Canvas）
- 拖拽上传 + 点击上传
- 图片尺寸限制提示（建议 ≤ 20MB）
- 自动缩放超大图片到合理尺寸（最大 4000px）
- 上传动画反馈

### 2. 滤镜引擎 (`engine/filterEngine.js`)
- **CSS Filter 层**：快速应用基础色彩调整
- **Canvas 叠加层**：暗角、渐变遮罩、胶片颗粒、
- **颜色矩阵层**：LUT 查找表、通道混合
- **滤镜强度**：在原始和处理后的像素之间做线性插值
- **防抖优化**：切换滤镜时使用 requestAnimationFrame 节流

### 3. 滤镜选择面板 (`ui/filterPanel.js`)
- 横向分类标签页
- 滤镜网格展示（带用户图片的缩略预览）
- 搜索功能（按名称、标签搜索）
- 收藏功能（localStorage 存储）
- 最近使用记录
- 虚拟滚动优化（处理 1000+ 列表）

### 4. 实时预览 (`ui/preview.js`)
- 双栏对比模式 / 单图模式切换
- 放大查看细节
- 全屏预览模式
- 移动端适配（上下分屏）

### 5. 导出模块 (`ui/export.js`)
- 格式：PNG（无损）、JPEG（可调质量 60-100%）
- 输出尺寸：原图尺寸 / 缩小选项
- 保留 EXIF 方向信息
- 下载文件名：`滤镜名_时间戳.png`
- 一键下载，无需确认对话框
